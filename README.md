# 💰 Tip Calculator

A modern, responsive, and user-friendly **Tip Calculator** built using **HTML5, CSS3, and Vanilla JavaScript**. This application helps users quickly calculate the **Tip Amount**, **Total Bill**, and **Amount Per Person** based on the entered bill amount, selected tip percentage, and number of people.

Designed with a clean UI, real-time calculations, proper input validation, and a fully responsive layout, this project is perfect for learning JavaScript fundamentals and showcasing in a web development portfolio.

---

## ✨ Features

- 💵 Enter the bill amount
- 🎯 Quick tip percentage buttons (5%, 10%, 15%, 20%, 25%)
- ✍️ Custom tip percentage input
- 👥 Split the bill among multiple people
- 🧮 Calculates:
  - Tip Amount
  - Total Bill
  - Amount Per Person
- ⚡ Instant calculation when values change
- ✅ Input validation
- 🚫 Prevents invalid bill amounts
- 🚫 Prevents number of people less than 1
- 💲 Currency formatted to two decimal places
- 🔄 Reset button to clear all inputs
- 🎨 Modern gradient UI with responsive design
- 📱 Mobile, tablet, and desktop friendly
- 🚀 Built without any external libraries

## 🚀 Installation & Usage

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Enter the **Bill Amount**.
5. Select a predefined tip percentage or enter a custom percentage.
6. Enter the number of people sharing the bill.
7. View the calculated:
   - Tip Amount
   - Total Bill
   - Amount Per Person
8. Click **Reset** to clear all values.
